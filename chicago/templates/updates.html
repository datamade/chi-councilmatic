Updates

Keep track of recent changes we have made to Chicago Councilmatic.

Aug 4, 2023: New legislation identifiers

As part of the Chicago City Clerk's update to a new Legislative Management System, any legislation that was currently active in June 2023 was given a new identifier. These identifiers are different than the ones they were given in the Legistar system. For example, bill O2023-2091 in Legistar is now O2023-0001686.

To account for this, Councilmatic considers the new identifiers to be the canonical ones and we have set up redirects for legislation using the old identifier. To help avoid confusion, we also list out what the old identifier was at the top of each legislation page, if it has one.


July 28, 2023: Using data from chicityclerkelms.chicago.gov

We are now using data from chicityclerkelms.chicago.gov to power Chicago Councilmatic.

We have, however, noticed several issues with how the data in the new system is structured. We have informed the Chicago City Clerk about these issues.

The biggest issue is that in the action history of legislation, the actors are often missing and wrong for newer matters, at least based on how it was done under the Legistar system.

For example, O2023-1020, https://chicityclerkelms.chicago.gov/Matter/?matterId=6FBDB317-5C09-EE11-8F6D-001DD809B965

  • The actor is missing for the "introduce" and "failed to pass" actions.
  • The actor is wrong for the "refer" action. The actor should be City Council, not the Committee on Transportation and Public Way.
  • The actor is wrong for the "recommend do not pass" action, it should be the Committee on Transportation and Public Way, not City Council.

This is pretty common with the recent legislation:

  • https://chicityclerkelms.chicago.gov/Matter/?matterId=359F7376-F509-EE11-8F6D-001DD809B965
  • https://chicityclerkelms.chicago.gov/Matter/?matterId=90303FC2-000A-EE11-8F6D-001DD809B578
  • https://chicityclerkelms.chicago.gov/Matter/?matterId=A5F30658-E30A-EE11-8F6D-001DD809B965

There are other challenges with the data. We are documenting the issues we run into in this public GitHub issue.


June 16, 2023: Chicago City Clerk launches new system

The Chicago City Clerk has launched a new Legislative Management System to replace Legistar. It can be reached at chicityclerkelms.chicago.gov.

While this system does have an API (api.chicityclerkelms.chicago.gov), it will take us some time to update our scrapers to bring in data from this new source. Until then, data will not be updated on Chicago Councilmatic. You can keep track of the progress made on updating the scrapers at github.com/opencivicdata/scrapers-us-municipal/tree/chicago_comprehensive/chicago.
